# VerifyPK

A tool to check Pakistani phone number and CNIC details from sim databases.

## Install

```bash
pip install -r requirements.txt
```

## Usage

**Check phone number:**
```bash
python VerifyPK.py -n 03XXXXXXXXX
```

**Check CNIC:**
```bash
python VerifyPK.py -c XXXXX-XXXXXXX-X
```
